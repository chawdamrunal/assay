# Code of Conduct

The Assay project adopts the [Contributor Covenant version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

## Summary

We pledge to make participation in this project a respectful, welcoming experience for everyone. We value technical rigor, constructive feedback, and clear communication. We do not tolerate personal attacks, discrimination, or sustained disruptive behavior.

## Scope

This Code of Conduct applies in all project spaces (issues, PRs, discussions, chat, conference talks, and any other forum where someone is representing the project) and in external spaces when an individual is acting as a project representative.

## Reporting

If you experience or witness behavior that violates this code, report it to:

- **Email:** `conduct@github.com/chawdamrunal/assay` (placeholder — will be updated before v1.0)

All reports are reviewed promptly and confidentially. The project maintainers will determine consequences in line with the Contributor Covenant's enforcement guidelines (see the full text linked above).

## The full text

The complete and binding text of the code we follow is the Contributor Covenant version 2.1. Read it at:

→ https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code. Enforcement actions may range from a private conversation through to permanent expulsion from project spaces, depending on severity and pattern. We follow the Contributor Covenant's recommended enforcement ladder.

If a maintainer themselves does not follow this code, action will be determined by the remaining maintainers.

---

By participating in the Assay community you agree to abide by this Code of Conduct.
